## Installation

Install rollcredits with npm

```bash
  npm install rollcredits
```

## Usage

Run `rollcredits` in the bash/terminal

```bash
rollcredits

// or

npx rollcredits
```

## Features

-   Clean Empty Folders
-   Organizes files (With Date)
